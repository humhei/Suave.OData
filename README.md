# Suave.OData.LiteDB
An Experimental OData Implementation in Suave With LiteDB.FSharp 
# Run Test
* cd Suave.OData.LiteDB.Tests
* dotnet restore
* dotnet run
# Debug Test In VSCode
* cd Suave.OData.LiteDB.Tests
* dotnet restore
* Press F5
